====
f.el
====

Modern API for working with files and directories in Emacs.

.. _cask: https://github.com/cask/cask

Installation
============

Add `f` to your `Cask`_ file:

.. code-block:: lisp

    (depends-on "f")

Then run `cask install` to install the package.
